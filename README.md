# 서울시 창업 흥행 가능성 예측 모델링
### Seoul Commercial District Startup Success Prediction with PCA, KMeans & XGBoost

서울시 열린데이터 광장의 상권 공공데이터를 기반으로, PCA 주성분 분석과 KMeans 군집화로 상권 특성을 파악하고 XGBoost 분류 모델로 업종-상권 조합의 창업 흥행 가능성을 예측한 빅데이터프로그래밍 팀 프로젝트입니다.

## 📖 Overview

자영업 창업의 대부분이 직관에 의존하는 현실에서, 데이터 기반 의사결정을 통한 창업 리스크 최소화를 목적으로 설계한 프로젝트입니다.

서울시 상권 데이터(인구, 점포, 매출, 상권 특성 등)를 수집·정제하여, PCA로 복잡한 상권 속성을 핵심 지표로 압축하고 KMeans로 상권 유형을 군집화하였습니다. 이후 파생 변수를 포함한 10개 피처를 구성하여 XGBoost 분류 모델로 업종-상권 조합의 흥행 여부를 예측하고, SHAP으로 판단 근거를 역추적하는 end-to-end 파이프라인을 구현하였습니다.

<br>

## ✨ Key Contributions

- 서울시 열린데이터 광장에서 제공하는 9개 상권 데이터를 업종·상권 필터링 후 병합하여, 상위 업종 30개 × 상권 15개 기준의 분석 데이터셋을 구축하였습니다.
- **PCA(n_components=2)** 를 적용해 6개 피처를 PC1(활동성·규모)·PC2(세부 특성·유형) 2개 주성분으로 압축하고, combined_pca 점수로 상권 활성도를 수치화하였습니다.
- **KMeans 군집화(k=3)** 를 통해 상권을 대형업무·중심상권 / 주거생활권 상권 / 소형동네·생활밀착상권으로 분류하고 지도 시각화하였습니다.
- **연령대별 TOP5 상권 분석** 을 통해 10대~60대 이상 각 연령대의 상위 방문 상권과 해당 상권의 최상위 매출 업종 조합을 도출하였습니다.
- **PCA 이동 경로 시각화(2021~2024)** 로 연도별 상권 변화 흐름을 추적하고, 4년간 변화량이 가장 큰 상위 10개 상권을 식별하였습니다.
- 비율·밀도·비교 기반 **파생 변수 8개** (순개업률, 유사업종경쟁도, 유동직장비율 등)를 설계하여 상권의 경쟁력·안정성·수요 밀도를 모델에 반영하였습니다.
- **XGBoost 분류 모델로 Accuracy 86%, Weighted F1 0.86** 을 달성하였으며, SHAP TreeExplainer로 흥행 판단 핵심 변수를 역추적하였습니다.
- XGBoost 예측 확률과 상권 위치 정보를 결합한 **흥행 확률 지도 시각화** 를 구현하였습니다.

## 🗂 Dataset

**서울시 열린데이터 광장 — 상권 관련 공공데이터 9종**

- 2020~2024년 상권 단위로 정리된 인구, 점포, 매출, 상권 특성 등 정량 공공데이터
- 추정매출, 점포수, 유동인구, 상주인구, 직장인구, 집객시설, 소득소비, 상권변화지표 등 포함
- 총 62개 업종 × 1,581개 상권 → 상위 업종 30개 × 상권 15개로 필터링 후 분석

<br>

## ⚙️ Pipeline

```
서울시 상권 공공데이터 (9종)
    │
    ▼
① 데이터 전처리
   ├─ 인코딩 자동 감지 (chardet) : 파일별 cp949/euc-kr 혼용 문제 자동 해결
   ├─ 업종·상권 필터링 (상위 업종 30개, 상권 15개 추출)
   ├─ 9개 테이블 병합 (상권_코드 기준 left join, 중복 행 drop_duplicates 처리)
   ├─ 범주형 수치화 : 상권변화지표 문자열 → 수치 매핑
   │     ('상권확장'→3, '다이나믹'→2, '정체'→1, '상권축소'→0)
   ├─ 비율 변수 정규화 : 폐업률·개업률 퍼센트 → 0~1 스케일 변환
   ├─ 이상치 처리 : np.inf / -np.inf → NaN 대체 후 결측 행 일괄 제거
   ├─ 좌표계 변환 : EPSG:2097 (TM중부) → EPSG:4326 (WGS84) (pyproj)
   └─ 표준화 (StandardScaler)
    │
    ├─ ② PCA 주성분 분석 (n_components=2)
    │       ├─ combined_pca 기반 상권 활성도 점수화
    │       ├─ KMeans 군집화 (k=3) → 상권/업종 유형 분류
    │       ├─ 연령대별 TOP5 상권 & 최상위 업종 도출
    │       └─ 연도별 상권 성장세 PCA 이동 경로 시각화 (2021~2024)
    │
    └─ ③ 흥행 예측 모델링
            ├─ 파생 변수 8개 생성 (비율·밀도·비교 기반)
            ├─ 흥행 여부 정의 (조합별 점포당 평균 매출 중앙값 기준)
            ├─ XGBoost 분류 모델 학습 (Accuracy 86%)
            ├─ SHAP TreeExplainer 기반 판단 근거 역추적
            └─ 업종별 상위 3개 상권 추천 + 흥행 확률 지도 시각화
```

## 🔍 주요 구현 포인트

**다종 공공데이터 통합 전처리**
9개 파일의 인코딩(cp949/euc-kr 혼용)을 `chardet`으로 자동 감지하여 로드하고, `상권_코드` 기준 left join으로 전체 테이블을 단계적으로 병합하였습니다. 범주형인 상권변화지표는 도메인 지식 기반으로 `상권확장(3) > 다이나믹(2) > 정체(1) > 상권축소(0)` 순서로 수치 매핑하였고, 지도 시각화를 위해 TM중부 좌표계(EPSG:2097)를 WGS84(EPSG:4326)로 직접 변환하는 파이프라인을 구성하였습니다.

**흥행 여부 정의의 공정성 확보**
절대적인 매출 기준이 아닌, 상권+업종 조합별 점포당 평균 매출의 전체 중앙값(60,690,469원)을 임계값으로 설정하였습니다. 이를 통해 업종 간 매출 규모 편차를 최소화하고, 클래스 불균형 없이 흥행(2,900건) / 비흥행(2,784건) 레이블을 균등하게 구성하였습니다.

**파생 변수 설계**
원시 변수만으로는 포착되지 않는 상권의 숨은 경쟁력을 반영하기 위해, 비율·밀도·비교 기반의 파생 변수 8개(순개업률, 유사업종경쟁도, 인구당집객시설수, 유동직장비율, 점포당거래건수, 소득대비지출, 폐업비율, 총생활인구)를 직접 설계하였습니다. SHAP 분석 결과, 이 파생 변수들이 모델의 핵심 판단 근거로 작동함을 확인하였습니다.

**연도별 상권 변화 시각화**
PCA 이동 경로 차트(2021~2024)를 통해 각 상권의 시계열 변화를 2차원으로 표현하였습니다. PC1(매출·유동인구·집객시설)과 PC2(직장인구·상주인구) 축의 이동 방향과 거리를 기준으로, 잠실역·강남역·삼성역 등 4년간 변화량이 가장 큰 상위 10개 상권을 정량적으로 식별하였습니다.
